# 아무튼 그 그거 - 다목적 디스코드 봇

![Python](https://img.shields.io/badge/Python-3.12+-3776AB?style=flat-square&logo=python&logoColor=white)
![discord.py](https://img.shields.io/badge/discord.py-2.0+-5865F2?style=flat-square&logo=discord&logoColor=white)


간편한 설정을 통해 서버 관리부터 게임까지 지원하는 다목적 디바이스 디스코드 봇입니다. 
Python 지식만 있다면 누구나 쉽게 기능을 확장할 수 있도록 설계되었습니다.

---

## 요약

이 봇은 개발자가 복잡한 환경 설정 없이도 바로 봇을 구동하고, 원하는 기능을 `cogs`에 추가하여 나만의 봇을 만들 수 있는 기반을 제공합니다.

- **자동화된 초기 설정:** `settings.py`를 통해 라이브러리 설치부터 토큰 설정까지 한 번에 끝납니다.
- **모듈형 구조:** `cogs` 폴더에 파일만 넣으면 자동으로 기능이 확장됩니다.
- **보안 중심:** 토큰과 관리자 정보는 철저히 로컬 환경 저장되어 외부 유출을 방지합니다.

---

## 기술 스택 및 구조

### 테크 스택
- **Language:** Python 3.12+
- **Library:** discord.py, rich, python-dotenv
- **Database:** SQLite3 (via aiosqlite)

### 프로젝트 구조
```text
root/
├── cogs/             # 봇의 기능 모듈 모음
├── .env.example      # 환경 변수 설정 템플릿
├── .gitignore        # GitHub 제외 파일 설정 
├── README.md         # 프로젝트 메인 가이드
├── requirements.txt  # 필수 라이브러리 목록
├── main.py           # 봇 실행 메인 파일
└── settings.py       # 초기 설정 및 환경 구축 스크립트
```

## 실행 및 설치 방법

### 1. 소스 코드 다운로드
```bash
git clone https://github.com/satellite4245/undefined.git
cd undefined
```

### 2. 초기 설정 (`settings.py`)
별도의 `pip install` 과정 없이 바로 `settings.py`를 실행하세요. 필요한 라이브러리 설치와 토큰/관리자 설정을 대화형 CLI로 진행합니다.

```bash
python settings.py
```
> **CLI 예시:**
> - 봇 토큰 입력 (비밀번호 형식으로 가려짐)
> - 관리자 디스코드 ID 설정
> - 봇의 활동 상태 및 명령어 접두사(`!`, `.` 등) 설정

### 3. 봇 실행
설정이 완료되었다면 봇을 가동합니다.
```bash
python main.py
```

## 사용 방법

1.  **명령어 동기화:** 봇을 처음 서버에 초대했다면, 관리자 계정으로 `!sync`를 입력하여 슬래시 명령어를 등록하세요.
2.  **기본 명령어:** `/test` 커맨드를 통해 봇의 작동 상태를 즉시 확인할 수 있습니다.
3.  **기능 확장:** 신규 기능을 만들고 싶다면 `cogs/` 폴더 내에 예시 파일을 참고하여 새로운 `.py` 파일을 생성하세요. 봇 재시작 시 자동으로 로드됩니다.
